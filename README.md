# WordPress API DB Tests

## Проверяемый функционал

Автотесты реализованы для следующих сущностей WordPress:

* записи;
* страницы;
* комментарии.

Основные операции:

* получение списка объектов;
* создание объекта через API;
* редактирование объекта через API;
* удаление объекта через API;

## Запуск WordPress

Необходимо поднять WordPress и MySQL:

```bash
docker compose up -d
```

После запуска WordPress доступен по адресу:

```text
http://localhost:8000
```

MySQL доступен на порту:

```text
3306
```
## Запуск тестов

Через Maven:

```bash
mvn clean test
```

Или через Maven-панель в IntelliJ IDEA:

```text
Maven → Lifecycle → test
```

## Результат запуска

Реализовано 13 автотестов:

* 4 теста для записей;
* 4 теста для страниц;
* 5 тестов для комментариев.

Успешный результат запуска.
